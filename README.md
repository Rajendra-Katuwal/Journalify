# Journalify🚀

Journalify is a modern and minimalistic journal app designed to help users document their thoughts, track their mood, and gain insights into their daily lives. The app focuses on simplicity, privacy, and an engaging user experience.

## Features
- **Secure Authentication**: User registration, email verification, and login.
- **Journal Entries**: Create, edit, and delete journal entries effortlessly.
- **Multimedia Support**: Attach photos and voice notes to entries.
- **View Modes**: Grid and calendar views for easy navigation.
- **Micro-Journaling**: Quick entry mode with predefined templates.
- **Privacy Control**: Keep entries private or share them publicly.
- **Search & Tags**: Filter entries using tags and keywords.
- **Reminders & Notifications**: Set daily or weekly journaling reminders.
- **Public Journals**: Share experiences with upvoting and commenting.

## Future Enhancements
- **AI-Powered Insights**: Sentiment analysis, smart tags, and automated summaries.
- **Mood Tracking & Trends**: AI-driven insights for self-reflection.
- **Smart Prompts**: AI-generated prompts based on past entries.
- **Customizable UI**: Personalized themes and font options.

## Tech Stack
- **Frontend**: React.js (Vite) with Tailwind CSS
- **Backend**: Django REST Framework (DRF)
- **Database**: PostgreSQL
- **Authentication**: JWT-based secure authentication

## Setup Instructions
1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/journalify.git
   cd journalify
   ```
2. Install frontend dependencies:
   ```sh
   cd frontend
   npm install
   ```
3. Start the frontend development server:
   ```sh
   npm run dev
   ```
4. Install backend dependencies:
   ```sh
   cd backend
   pip install -r requirements.txt
   ```
5. Run the backend server:
   ```sh
   python manage.py runserver
   ```
